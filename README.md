Title: From Data to Decision: Customer Churn Prediction using Neural Networks

Overview

This project focuses on analyzing customer behavior and predicting churn in a telecommunications dataset. The objective is not only to build a predictive model but also to interpret results and derive actionable business insights.

Contents :
The Jupyter Notebook contains the complete workflow, including data preprocessing, exploratory data analysis (EDA), model development, and evaluation.
The report summarizes key findings, model performance, and business decisions derived from the analysis.
Visualizations are included to support insights and interpretations.
Highlights :
Identified key drivers of churn such as contract type, tenure, and monthly charges
Built a neural network model achieving ~81% accuracy
Highlighted the importance of recall in churn prediction
Proposed actionable business strategies for customer retention

Outcome :

The project demonstrates how data-driven insights can be used to predict customer churn and support strategic decision-making.
